README file 

Was ist unser Projekt?
Unser Projekt stellt eine Website zur Verfügung, die es möglich macht für personen, die sich mit ihrem Google Account anmelden, in einem offenen Chat Nachrichten zu schreiben und zu empfangen

Wie weit ist unser Projekt?
Unser Projekt befindet sich momentan noch in der Bearbeitung, stellt aber die Grundliegenden Funktionen bereits zur Verfügung. Aber vorallem das Design muss noch anschaulicher gestaltet werden und die Möglichkeit einzelnen Personen Texte zu schreiben muss hinzugefügt werden.
